Adds a small floating dashboard button on the frontend for logged-in users who have disabled the WordPress toolbar.

== Description ==

WPAdmin Button shows a round button on the frontend that expands into a menu of WordPress admin shortcuts — including a contextual "Edit current page" link when viewing an editable post or page.

Administrators choose which shortcuts appear and their order, which roles see the button, and whether it sits at the bottom-left or bottom-right of the page, all from Tools > WPAdmin Button. Each shortcut is automatically hidden from any user who lacks permission to use it.

From their own profile page, each user chooses when the button appears — automatically (only when the WordPress toolbar is hidden), always, or never — and can hide any shortcuts they don't want to see.

When WordPress reports pending updates, users with update permissions also see a small red badge on the button. The badge links directly to Dashboard > Updates.

The button and its menu use the logged-in user's Administration Color Scheme.

== Updates ==

Plugin updates are served from GitHub Releases.
